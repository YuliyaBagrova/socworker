# SocWorker

Система управления информацией о социальных работниках и их подопечных.

## Описание проекта

SocWorker — это веб-приложение на Django, предназначенное для систематизации информации о социальных работниках и назначенных им подопечных. Система позволяет эффективно управлять данными, отслеживать взаимодействие между работниками и подопечными, а также генерировать отчеты для анализа работы социальной службы.

## Основной функционал

- **Управление пользователями**: Регистрация и авторизация пользователей системы
- **Управление социальными работниками**: Ведение базы данных социальных работников с полной информацией о них
- **Управление подопечными**: Регистрация и ведение информации о подопечных, назначенных социальным работникам
- **Связь работник-подопечный**: Назначение подопечных конкретным социальным работникам
- **Генерация отчетов**: Создание различных отчетов для анализа работы и статистики

## Технологический стек

- **Backend**: Django (Python)
- **База данных**: MySQL
- **Аутентификация**: JWT (JSON Web Tokens)
- **API**: Django REST Framework (предполагается)

## Структура базы данных

### Таблицы:

1. **Пользователи (Users)**
   - Учетные записи пользователей системы
   - Информация для авторизации и управления доступом

2. **Социальные работники (Social Workers)**
   - Персональная информация о социальных работниках
   - Контактные данные
   - Статус и должность

3. **Подопечные (Wards)**
   - Информация о подопечных
   - Персональные данные
   - Связь с назначенным социальным работником

## Установка и запуск

### Требования

- Python 3.8+
- MySQL 5.7+ или MySQL 8.0+
- pip (менеджер пакетов Python)

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Настройка базы данных

1. Создайте базу данных MySQL:
```sql
CREATE DATABASE socworker CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

2. Настройте подключение в `settings.py`:
```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'socworker',
        'USER': 'your_username',
        'PASSWORD': 'your_password',
        'HOST': 'localhost',
        'PORT': '3306',
    }
}
```

### Применение миграций

```bash
python manage.py makemigrations
python manage.py migrate
```

### Создание суперпользователя

```bash
python manage.py createsuperuser
```

### Запуск сервера разработки

```bash
python manage.py runserver
```

Приложение будет доступно по адресу: http://127.0.0.1:8000/
